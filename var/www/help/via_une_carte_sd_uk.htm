Via an SD

Before you begin make sure that the multispectral sensor is connected to the sunshine sensor.

1.Connect your USB key to a computer and make sure that it contains no files in .plf format.

2.Go to the Sequoia help page: www.parrot.com/uk/support/

3.Download the update file available in .plf format.

4.Copy the file to the root directory on your SD Card without putting it in a folder or renaming it.

5.Insert the SD card in the sunshine sensor.

6.Turn on Sequoia.

> The update runs automatically. During the update the multispectral sensor light flashes orange (Diagram J). Once the update is complete the light turns green.
